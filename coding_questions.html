Question 1 Vector Operations with Vectorization

Problem Statement:

Write a class with methods to perform vector operations including:

  • Addition of two vectors
  • Dot product of two vectors
  • Normalization of a vector

Print their results for the following vectors:

v1 = Vector([1, 2, 3]) v2 = Vector([4, 5, 6])
üí° Solution:

                import numpy as np

                class Vector:
                    def __init__(self, elements):
                        # Initialize vector using numpy array for vectorization
                        self.elements = np.array(elements)

                    def addition(self, other):
                        # Vector addition using numpy vectorization
                        result = self.elements + other.elements
                        return Vector(result)

                    def dot_product(self, other):
                        # Dot product using numpy's dot function
                        return np.dot(self.elements, other.elements)

                    def normalize(self):
                        # Normalize vector (divide by magnitude)
                        magnitude = np.linalg.norm(self.elements)
                        if magnitude == 0:
                            return Vector(self.elements)
                        normalized = self.elements / magnitude
                        return Vector(normalized)

                    def __str__(self):
                        # String representation for printing
                        return f"Vector({self.elements})"


                # Create vectors
                v1 = Vector([1, 2, 3])
                v2 = Vector([4, 5, 6])

                # Perform operations and print results
                print("Vector 1:", v1)
                print("Vector 2:", v2)
                print("\n--- Operations ---")

                # Addition
                result_add = v1.addition(v2)
                print(f"Addition: {result_add}")

                # Dot Product
                result_dot = v1.dot_product(v2)
                print(f"Dot Product: {result_dot}")

                # Normalization
                result_norm_v1 = v1.normalize()
                result_norm_v2 = v2.normalize()
                print(f"Normalized v1: {result_norm_v1}")
                print(f"Normalized v2: {result_norm_v2}")

  • Vectorization: Uses NumPy arrays for efficient element-wise operations
  • Addition: Element-wise addition: [1+4, 2+5, 3+6] = [5, 7, 9]
  • Dot Product: Sum of element-wise multiplication: (1√ó4) + (2√ó5) + (3√ó6) = 32
  • Normalization: Divides each element by the vector's magnitude (‚àö(1¬≤+2¬≤+3¬≤) = ‚àö14 ‚âà 3.742)
  • Benefits: NumPy vectorization is 10-100x faster than loops for large arrays
